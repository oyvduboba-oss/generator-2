# PECS Card Generator

Простой мини‑проект на **HTML, CSS и JavaScript** без сборки и без зависимостей.

Открывается локально через браузер и позволяет:

- создавать одну PECS / AAC карточку;
- загружать изображение;
- задавать подпись и язык;
- выбирать визуальный стиль (Premium Montessori или Photorealism);
- выбирать расположение подписи;
- выбирать AAC‑категорию, от которой зависит цвет рамки;
- выбирать размер карточки (Small / Medium / Large / Class);
- видеть live preview;
- экспортировать карточку в PNG;
- сбрасывать настройки кнопкой Reset.

## Структура проекта

- `index.html` — разметка страницы, подключение шрифта Nunito, CSS и скрипта.
- `style.css` — оформление интерфейса и карточки, CSS‑переменные для цветов и размеров.
- `script.js` — логика работы приложения: состояние, live preview, загрузка картинки, экспорт в PNG, Reset.

## Как запустить

1. Склонируйте репозиторий или скопируйте папку с файлами.
2. Откройте файл `index.html` двойным кликом (или через «Открыть с помощью → Браузер»).
3. Приложение сразу готово к использованию, сервер не нужен.
